## Tecnologias:

- [Adonis.js](https://adonisjs.com/) 🚀

### Brief description

- Api for the purpose of simulating the action of crediting and debiting from a bank account

### Modulos:

- User Creation 👱
- Banks Creation 📝
- Account Creation 💰
- Crediting Method 💱
- Debit Method 💱

### Routes:

- /created
- /users/:id
- /users
- /login
- /api/v1/banks (with all the methods)
- /api/v1/accounts (with all the methods)
